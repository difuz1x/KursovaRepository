# TaskManager

Легка React + Vite аплікація для керування домашніми завданнями.

Коротко:
- Додавайте завдання з пріоритетом та дедлайном.
- Фільтрація за статусом, сортування за датою/пріоритетом.
- Графіки пріоритетів та статусу (Chart.js).
- Підтвердження при видаленні та очищенні, Undo для видалених/очищених завдань.

Запуск локально:

## Імпорт/експорт задач

- У `public/sample-tasks.json` є приклад для тестування імпорту.
- UI дозволяє вибирати файл JSON, переглядати прев'ю, бачити дублікати та або зливати, або замінювати список задач.
- Імпортовані дані валідовуються схемою `zod` — невірні задачі показуються у прев'ю з помилками.

---

## Ключова функціональність

- Додавання задач з полями: `title`, `description`, `priority` (low|medium|high), `dueDate` (ISO), `estimatedMinutes`.
- Час виконання: окремі поля годин/хвилин або число + одиниця; швидкі кнопки для додавання часу.
- Таблиця задач: показує стовпець "Час" (формат: X г Y хв), сортування, іконки-попередження для дат з перевантаженням.
- Перевірка по даті: обчислення сум часу по датам, банер якщо сума > 12 годин, блокування додавання якщо буде > 24 години.
- Локальне сховище (`localStorage`) з логікою міграції для старих схем даних.

---

## Структура даних (TaskType)

- `id: string`
- `title: string`
- `description?: string`
- `priority: 'low' | 'medium' | 'high'`
- `dueDate?: string` (ISO)
- `estimatedMinutes?: number`
- `isCompleted: boolean`
- `createdAt?: string` (ISO)

Імпорт нормалізується у цю форму за допомогою `zod`.
